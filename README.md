# Proyecto Alura "AntiumGear"

Este repositorio contiene el desarrollo del agente de atención al cliente **AntiumGear**, creado como parte del Challenge Alura.  
El objetivo es construir un chatbot capaz de responder preguntas frecuentes de la tienda, utilizando documentos internos como:  
- Política de privacidad  
- Política de reembolso y devoluciones  
- Preguntas frecuentes (FAQ)  
- Guía de envíos y entregas  
- Términos y condiciones  

## 🎯 Objetivos del Challenge
- Comprender los requisitos y criterios de evaluación del Challenge Alura Agente.  
- Definir una base de conocimiento utilizando documentos en formato PDF o CSV.  
- Diseñar y desarrollar un agente inteligente capaz de responder preguntas en lenguaje natural.  
- Organizar y documentar un proyecto profesional en GitHub.  
- Elaborar un README completo con arquitectura, instrucciones de uso y evidencias del proyecto.  
- Implementar una solución funcional utilizando herramientas y modelos de inteligencia artificial.  
- Realizar el despliegue de la aplicación en la nube utilizando Oracle Cloud Infrastructure (OCI).  

## 📂 Estructura del proyecto
- `docs/` → Documentación y políticas (Privacidad, Reembolsos, FAQ, Envíos, Términos).  
- `src/` → Código fuente del agente inteligente.  
- `tests/` → Pruebas y ejemplos de uso.  
- `README.md` → Documentación principal del proyecto.  

## 🚀 Futuras funcionalidades
- Integración con la nube para despliegue en OCI.  
- Ejemplos de interacción con clientes.  
- Capturas de pantalla y video demostrativo.
